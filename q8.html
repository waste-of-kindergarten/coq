除了这种定义外，我们仍然可以使用归纳的方法对偶数进行定义。

  • 0 是偶数
  • 如果n是偶数，那么S (S n)也是偶数
Inductive ev : nat -> Prop :=
| ev_0 : ev 0
| ev_SS (n:nat) (H : ev n) : ev (S (S n)).

Inductive 定义了一个从nat到Prop的映射，ev内含两个证据构造子(evidence constructors) ev_0 以及 ev_SS.

将定义中涉及到的类型均展示出来:

关于ev_SS:

ev_SS 本身接收两个参数——n和H,类型分别为nat和ev n(注意不是参数可以接收nat 和 ev n,而是参数所属的类型). ev_SS 本身的类型相当于一个蕴含式，从程序上讲是一个函数，允许接收两个参数，接收参数后变成高阶函数，类型也从全称命题变成了一个针对特定值的结论。

区分： 在Inductive 定义中，类型构造子冒号左侧的参数为形参(parameter)，右侧叫做索引(index)或注解(annotation)，如ev定义的时候就使用了索引，这使得ev 后面可以接收特定的nat类型的值。

在证明中使用证据

对于ev的定义，其不仅说明ev_0和ev_SS是构造偶数证明证据的方式，同时也说明是ev中构造的全部方式。

这样我们可以像分析归纳定义的数据结构一样分析形如ev n的假设。可以使用归纳(induction)和分类讨论(case analysis)来进行证明。

对证明进行反演（inversion）

通过destruct 可以对证据进行解构。

Theorem ev_inversion :
forall n : nat, ev n -> (n = 0) \/ (exists n', n = S (S n') /\ ev n').
Proof.
intros n E.
destruct E as [ | n' E'].
(*E = ev_0 : ev 0*)
- left. reflexivity.
(*E = ev_SS n' E' : ev (S (S n'))*)
- right. exists n'. split. reflexivity. apply E'.
Qed.

这里面的destruct对前提E进行了解构，由于E本身含有类型ev n.而ev又定义了两个证据构造子，所以destruct自然地将E解构成ev_0 和 ev_SS n' E'.

考虑之前学习自然数时对变量n的解构，实际上原理是相同:n类型时nat,而nat定义了两个构造子，所以destruct n将n非为了0和S n'.

然而，destruct 不是非常智能，有时需要一些辅助，例如下面这个定理的证明:

Thereom evSS_ev : forall n,
ev (S (S n)) -> even n.
Proof.
intros n E.
destruct E as [| n' E'] eqn : E1.
- (*E = ev_0 : ev 0*)
(* 这显然是不可能的，但条件不足无法证明下去 *)
Abort.

(*使用remember 进行变量替换*)
Theorem evSS_ev : forall n,
ev (S (S n)) -> even n.
Proof.
intros n E.
remember (S (S n)) as k.
(*设 k = (S (S n))*)
destruct E as [| n' E'].
- (*E = ev_0 *)
(*0 = k = S (S n) 这显然不可能，且可以证明矛盾*)
discriminate Heqk.
- (*E = ev_S n' E'*)
injection Heqk as Heq.
rewrite Heq in E'. 
apply E'.
Qed.

或者也可以应用ev_inversion将evSS_ev 中的(S (S n))与ev_inversion中的n匹配。这样也可以解构前提。

Theorem evSS_ev1 : forall n ,ev (S (S n)) -> ev n.
Proof.
intros n H.
apply ev_inversion in H.
destruct H.
- discriminate H.
- destruct H as [n' [ Hnm Hev]].
injection Hnm as Heq.
rewrite  Heq.
apply Hev.
Qed.

Coq提供一个更为强大的策略inversion,对于前提是等式，该策略可以自动使用discriminate和injection,同时也会使用intros和rewrite对前提进行化简;对于具有归纳定义的前提，该策略也可以对其进行解构，生成的子目标中矛盾的情况将被忽略，并保留其余子命题，我们只需要对其余的命题进行证明即可得证原命题。

Theorem evSS_ev2 : forall n,
ev (S (S n)) -> ev n.
Proof.
intros n E.
inversion E as [| n' E' EQ].
apply E'.
Qed.

对证据进行归纳

对证据使用induction与对数据使用具有相同的行为——引导Coq对每个证据构造子生成一个子目标，同时提供了归纳假设。

Lemma ev_even : forall n,
ev n -> even n.
Proof.
intros n E.
induction E as [| n' E' IH].
- (* E = ev_0 *)
exists 0. reflexivity.
- (* E = ev_SS n' E'*)
(*IH : even n'*)
destruct IH as [k' Hk'].
(* 对even的定义进行解构
exists k' : n' = double k' 
*)
rewrite Hk'.
exists (S k').
reflexivity.
Qed.

归纳关系

ev定义了nat的一个子集，可以视为是一个'性质'。而有两个参数的命题就可以视为一个'关系'，定义了一个序对集合。

例如，小于等于:

Inductive le : nat -> nat -> Prop :=
| le_n (n :nat) : le n n 
| le_S (n m : nat) (H : le n m) : le n (S m).
Notation "m <= n" := (le m n).

使用le_n和le_S构造子，可以证明关于le的事实。

Theorem test:
2 <= 4 /\ ~(2 >= 4).
Proof.
split.
apply le_S.
apply le_S.
apply le_n.
unfold not.
intros.
inversion H.
inversion H2.
inversion H5.
Qed.

小于关系的定义，可以使用le定义，也可以单独进行归纳定义。

Definition lt(n m : nat) := le (S n) m.

Definition lt1 : nat -> nat -> Prop :=
| lt_n (n : nat) : lt n (S n)
| lt_S (n m : nat) (H : lt n m) : lt n (S m).

Notation "m < n" := (lt m n).
