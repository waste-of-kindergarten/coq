Coq笔记 <COIC vs 集合论 小记>

Coq的逻辑核心是COIC,即归纳构造演算(Calculus of Inductive Construction),其在很多方面不同于数学家用来写下的精确而严谨的定义和证明的形式化系统。例如ZFC系统——一个数学对象可同时属于不同的集合;在Coq中一个项最多只属于一个类型。

在某些情况下，直接将数学论证翻译到Coq很繁琐甚至不可能做到，此时需要引入新的公理来丰富逻辑核心。

函数外延性

函数外延性指，对于任意两个函数，只要它们对相同的输入产生相等的结果，那么它们就被认为相等。

(∀x,fx=gx)−>f=g(\forall x , f x = g x) -> f = g(∀x,fx=gx)−>f=g

这是一个非常重要的原理，但是并不在Coq基本公理之内，因此某些很显然的命题是不能证明的。

为此引入一个新的公理，将函数的外延性添加Coq的核心逻辑系统中。

Axiom functional_extensionality : forall {X Y : Type} {f g : X -> Y},
(forall (x:X) , f x = g x) -> f = g.

公理的添加应当尤为小心，因为有可能导致系统不一致

命题和布尔值

Coq核心语言具有'可计算性'，即它所表达的所有函数都是可计算且完全的，目的是为了能从Coq开发的代码中提取出可执行程序。Coq中的Prop没有一种通用的情况分析操作来确定任意给定的命题是否为真，否则就能够写出不可计算的函数。

尽管一般的不可计算性质无法表述为布尔计算，但值得注意的是，很多 '可计算的'性质更容易通过 Prop 而非 bool 来表达，因为在 Coq 中定义递归函数中会受到很大的限制。

不过，通过布尔值来陈述事实会带来一点重要的优势，即通过对 Coq 中的项进行计算可以实现一些自动推理，这种技术被称为'互映证明（Proof by Reflection）'。

<软件基础>中给出的示例：

将Prop转化成bool

Theorem even_bool_prop : forall n,
evenb n = true <-> even n.
(*具体证明这里不予书写，只是用来说明定理内容*)
Admitted.

Example even_1000 : even_1000.
Proof.
unfold even.
exists 500.
reflexivity.
Qed.

Example even_1000' : evenb 1000 = true.
Proof. reflexivity. Qed.

Example even_1000'' : even 1000.
Proof.
apply even_bool_prop.
reflexivity.
Qed.

对于更大的证明，优势会凸显出来。在四色定理中，人们使用互映讲几百种不同的情况规约成了一个布尔计算。

将bool转化成Prop

Theorem eqb_eq : forall n1 n2 : nat,
n1 =? n2 = true <-> n1 = n2.
(*具体证明这里不予书写，只是用来说明定理内容*)
Admitted.

Lemma plus_eqb_example : forall n m p : nat ,
n =? m = true -> n + p =? m + p = true.
Proof.
intros n m p H.
rewrite eqb_eq in H.
rewrite H.
rewrite eqb_eq.
reflexivity.
Qed.

经典逻辑和构造逻辑

在Coq的证明中，也存在无法证明的命题:

Definition excluded_middle := forall P : Prop,
P \/ ~ P.

由于缺乏必要的信息，无法在函数内部机械地确定P是否成立。但是可以借助与布尔值互映实现。

Theorem restricted_excluded_middle : forall P b,
(P <-> b = true) -> P \/ ~ P.
Proof.
intros P [] H.
-left. rewrite H. reflexivity.
-right. rewrite H. intros contra. discriminate contra.
Qed.

一般的排中律在 Coq 中默认并不可用，因为它是常见的逻辑系统（如 ZFC）中的标准特性。尽管如此，不假设排中律的成立仍有其独特的优点： Coq 中的陈述可以构造出比标准数学中同样陈述更强的断言。特别是， 当存在 ∃ x, P x 的 Coq 证明时，我们可以直接给出一个使 P x 得证的值 x。换言之，任何关于存在性的证明必定是'构造性'的。
像 Coq 一样不假设排中律成立的逻辑系统被称作'构造逻辑'。
像 ZFC 这样更加传统的，排中律对于任何命题都成立的逻辑系统则被称作'经典逻辑'。
