关于导入文件

由于需要使用之前的定义，我们需要将上一节的文件导入，Coq中文件导入需要将脚本进行编译，对于CoqIDE用户来说，这非常容易。

上一节的脚本文件q1.v,操作如下

  • 在脚本下创建一个文件名为_CoqProject,内容为
    -Q . Coq

其中'.'表示当前目录，Coq是当前目录的文件夹名称（逻辑目录）

  • 用CoqIDE打开q1.v,在菜单中找到Compile栏，并点击Compile Buffer

可以看到文件夹中出现了q1.vo文件，这是q1.v的编译版本，在q2.v中添加如下语句

    From Coq Require Export q1.

按Ctrl-End看到该行被标注为绿色，说明导入成功。

附：在使用IDE时，可能出现编译之后导入失败的情况，我遇到的问题是Error:Unable to locate library q1,《Software Foundation》中给出的提示是CoqIDE编译的代码和在指令行用的'coqc'编译不一致

最无脑的方式是更换新版本，经过试验8.15Beta版本可以成功编译并导入

归纳法证明

在上一节，我们证明了Theorem plus_0_n':

    Theorem plus_0_n' : forall n:N , plus O n = n.

这个定理由定义立即可得，现在考虑Theorem plus_n'_0:

    Theorem plus_n'_0 : forall n:N , plus n O = n.

这似乎是显然的，但是利用上节已知的定义和证明方法（化简，改写和分类讨论）是不能得出完整证明的，似乎是一个无限下去的死循环。

Coq支持数学归纳法，这使得上述定理得以证明。

  • 证明 plus O O = O
  • 证明对于任何n',若 plus n' O = n',则 plus (S n') O = S n'
  • 最后得出，命题对于任意自然数都成立
    Proof.
    intros n.
    induction n as [| n' IHn'].
    - reflexivity.
    - simpl. rewrite <- IHn'. reflexivity.
    Qed.

归纳使用关键字induction,将命题拆分成两个子目标 'as [| n' IHn']' ,'|'将两个子目标的变量命名分开，第一个子目标没有新变量引入，于是为空，第二个子目标中归纳假设变量命名为n',归纳假设的表达式命名为IHn'.

证明过程分析

  • 将n添加到上下文中，记为n
  • 将n进行数学归纳，原命题的证明分解为两个子目标，第二个子目标的变量和归纳假设前提分别记为n'和IHn'
  • 第一个子目标： 直接化简
  • 第二个子目标： 化简，使用IHn'从右向左改写，并进行化简

(提示：induction策略被应用到包含量化变量的目标中时，它会自动将需要的变量移到上下文，因而intros n.并不是必要的)

    Theorem plus_n'_1: forall n:N,
    plus n (S O) = S n.
    Proof.
    induction n as [| n' IHn'].
    -reflexivity.
    -simpl. rewrite IHn'. reflexivity. Qed.

    Theorem mult_n'_0: forall n:N, mult n O = O.
    Proof.
    induction n as [| n' IHn'].
    -reflexivity.
    -simpl. rewrite IHn'. reflexivity.
    Qed.

证明中的证明

在非形式化数学中，对于大的证明往往需要拆解，如果能在需要时对琐碎而缺乏普遍性的事实进行简单陈述并立即证明所需'子定理'，将会带来很多方便。

Coq中允许使用内联assert进行子定理的声明。

    Theorem mult_0_plus' : forall n m :N,
    mult (plus O n) m = mult n m.
    Proof.
    intros n m.
    assert (H: plus O n = n). {reflexivity.}
    rewrite -> H.
    reflexivity.
    Qed.

assert 策略引入两个目标，第一个目标是子定理plus O n = n,'{}'中包含的是子定理的证明；在获得子定理H后，第二个目标使用H对目标进行改写完成证明。

(提示：assert (H: plus O n = n).可以改用as进行命名，即assert (plus O n = n) as H.)
